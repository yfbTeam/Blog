Mongodb关联查询：
Mongodb3.2之后（2015年中旬左右），版本更新之后，增加了一个Lookup，叫做左关联




//运行以下语句时，注意删除注释
db.product.aggregate([//
     {
       $lookup:
         {
           from: "orders",//表名，是我们需要关联查询的表
           localField: "_id",//表需要关联的键
           foreignField: "pid",//【orders】的matching key
           as: "inventory_docs"//别名
         }
    }
 ])
